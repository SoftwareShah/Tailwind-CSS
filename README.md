## 📚 What I Learned & Implemented

### 🔤 Typography
- Font families: `font-sans`, `font-serif`, `font-mono`
- Font weights: `font-bold`, `font-black`, `font-extrabold`
- Font sizes & line heights: `text-4xl`, `text-xl/8`
- Text alignment: `text-center`, `text-right`, `text-justify`, `text-start`, `text-end`
- Text decoration: `underline`, `overline`, `line-through`, `decoration-wavy`, `underline-offset-*`
- Text indent: `indent-8`, `-indent-7`, `indent-[50%]`
- Text overflow: `line-clamp-2`, `wrap-anywhere`, `hyphens-auto`
- Whitespace control: `whitespace-pre`, `whitespace-pre-wrap`
- Letter spacing: `tracking-wide`
- Vertical alignment: `align-baseline`, `align-top`
- OpenType features: `ordinal`, `diagonal-fractions`

### 🎨 Colors & Backgrounds
- Utility colors: `bg-pink-300`, `bg-cyan-400`, `bg-purple-500`, `bg-gray-900`
- Text colors with opacity: `text-blue-600`, `dark:text-blue-600/50`
- Responsive color variants: `md:text-green-600`, `sm:text-black`
- Accent colors: `accent-blue-600`, `accent-red-600`
- Caret color: `caret-red-600`

### 📦 Spacing & Sizing
- Padding & margin: `p-*`, `px-*`, `py-*`, `m-*`, `ml-*`, `mt-*`
- Width & height: `w-full`, `w-96`, `h-auto`, `size-16`
- Max width: `max-w-sm`, `max-w-3xl`
- Custom values: `w-[95%]`, `indent-[50%]`

### 🃏 Borders & Shadows
- Border radius: `rounded`, `rounded-xl`, `rounded-2xl`, `rounded-full`
- Border styles: `border`, `border-gray-700`, `border-4`
- Box shadows: `shadow-lg`, `shadow-xl`, `shadow-2xl`
- Outline on hover: `hover:outline`, `hover:outline-blue-600`, `hover:outline-4`

### 📐 Flexbox
- `flex`, `flex-col`, `flex-row`, `flex-wrap`
- Justify & align: `justify-center`, `items-center`, `items-start`
- Gap: `gap-3`, `gap-6`, `space-x-28`
- Flex basis: `basis-32`, `basis-64`, `basis-lg`
- Shrink: `shrink-0`
- Responsive flex: `md:flex-row`

### 🔲 CSS Grid
- `grid`, `grid-cols-3`, `grid-cols-4`, `grid-rows-4`
- Auto flow: `grid-flow-col`
- Spanning: `row-span-4`, `col-span-2`
- Content alignment: `content-stretch`

### 🖼️ Images & Filters
- Object fit: `object-cover`
- Hue rotation: `hue-rotate-140`, `hue-rotate-180`, `-hue-rotate-15`
- Rounded images: `rounded-full`, `rounded-2xl`

### 🌀 Mix Blend Modes
- `mix-blend-screen`
- `mix-blend-lighten`
- `mix-blend-color-dodge`
- `mix-blend-exclusion`
- `mix-blend-difference`
- `mix-blend-luminosity`

### 🎭 Transforms & Transitions
- Rotate: `rotate-15`, `hover:rotate-0`
- Translate: `translate-y-2.5`, `hover:translate-y-0`
- Transition: `transition-transform`, `duration-500`, `ease-in-out`

### 🖱️ Interactivity & States
- Hover states: `hover:bg-indigo-500`, `hover:bg-gray-600`
- Focus states: `focus:outline-none`, `focus:ring-2`, `focus:ring-pink-500`, `focus:border-yellow-300`
- User select: `select-none`
- Pseudo-elements: `after:content-['_↗']`

### 📋 Lists
- `list-disc`, `list-decimal`, `list-inside`, `list-outside`
- Custom list image: `list-image-[url(...)]` with inline SVG

### 📊 Tables
- `border-collapse`, `caption-bottom`
- Responsive table with styled `thead`, `tbody`, `th`, `td`

### 🧩 Components Built
- ✅ Profile card with avatar, badge, Follow/Message buttons
- ✅ Email input fields with custom focus rings
- ✅ Textarea with floating Post button
- ✅ Mix-blend-mode visual demos
- ✅ Hue-rotate image gallery
- ✅ Responsive profile bio card with divider
- ✅ Grid layouts with row/col spanning
- ✅ Flex basis layout demos

## 🏃 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm start
```

Open `http://localhost:5173` in your browser.
